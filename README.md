# Ralph-Matthew-Punzalan

Personal portfolio for R4lph Matthew Punzalan — front-end development, UI/UX design, and featured project work.

## Setup

```bash
npm install
npm run dev
```

## Contact form (Supabase)

Copy `.env.example` to `.env.local` and add your Supabase URL and anon key. Run `supabase/contact_messages.sql` in your Supabase SQL editor.
